# RD-INFRO-TECHNOLOGY
Data Analysis Internship – NIFTY 50 Project
## Project Title
NIFTY 50 Stock Market Data Analysis (2000–2025)

## Objective
To analyze historical NIFTY 50 data and derive insights using Python and Power BI.

## Tools Used
- Python (Pandas, NumPy, Scikit-learn, Matplotlib)
- Power BI
- MS Word
- GitHub

## Tasks Completed
- Data Collection
- Data Cleaning
- Data Transformation
- Exploratory Data Analysis
- Statistical Analysis
- Predictive Modeling
- Dashboard Creation
- Report Generation & Business Insights

## Key Insights
- Long-term upward trend in NIFTY 50
- Moderate volatility
- Moving average smooths fluctuations
- Interactive dashboard improves analysis
